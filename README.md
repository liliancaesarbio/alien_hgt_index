AI_hU_formula
Command-line formulas used to calculate the Alien Index (AI) and HGT index (hU) from BLAST/DIAMOND search results, following the definitions in Gladyshev et al. (2008) and Boschetti et al. (2012).

diamond_taxa_ann.py
Python script that parses DIAMOND/BLASTP output in format 6 and retrieves the taxonomic annotation of each hit using the NCBI taxonomy database.
